🔧 Limpar Auth Lock

Use este utilitário para limpar locks de autenticação travados

Instruções:

  1. Clique em "Limpar Tudo" para remover todos os dados do localStorage
  2. Feche TODAS as abas do localhost:5173
  3. Abra uma nova aba e acesse http://localhost:5173/login
  4. Faça login normalmente
